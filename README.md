Installing
----------

Install EasyPaint using the commands, if you use CMake:

    cd ./EasyPaint
    cmake -DCMAKE_INSTALL_PREFIX=/usr
    make
    make install

Install EasyPaint using the commands, if you use qmake:

    cd ./EasyPaint/sources
    lrelease easypaint.pro
    qmake
    make
    make install
